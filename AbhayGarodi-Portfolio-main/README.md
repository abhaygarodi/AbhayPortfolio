# Personal Portfolio Website

## Overview
This is my personal portfolio website designed to showcase my professional profile, technical skills, and project work. The portfolio provides a clear overview of my background, highlights key projects, and offers an easy way to get in touch through a functional contact form.

The website is built with a focus on clean UI, responsiveness, and performance, ensuring an optimal experience across desktop, tablet, and mobile devices.

---

## Features
- Responsive and modern user interface  
- Dedicated sections for About, Skills, Projects, and Contact  
- Interactive project showcase  
- Contact form with email functionality  
- Clean navigation and structured layout  

---

## Technologies Used

### Frontend
- React.js  
- HTML5  
- CSS3  
- JavaScript  

### Email Service
- EmailJS (for contact form email integration)

## Projects
The portfolio includes selected projects that demonstrate:
- Practical implementation of frontend technologies  
- Clean and reusable code structure

Each project section contains a brief description and relevant details.

## Contact
The contact section allows visitors to directly reach out via email using EmailJS. Messages submitted through the form are securely delivered without requiring a backend server.
